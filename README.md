# Log Monitoring and Analysis using Graylog

## Description
This project focuses on basic log monitoring using Graylog, a centralized log management and SIEM tool. The aim is to understand log aggregation, monitoring, and basic analysis.

## Tools Used
- Graylog (Basic setup)
- Linux system logs
- Network logs (optional)

## What I Did
- Set up Graylog for centralized log monitoring.
- Collected logs from sample sources.
- Analyzed logs to detect unusual system activity or anomalies.
- Learned how to create basic dashboards for monitoring.

## Learning Outcomes
- Understanding of centralized log collection and analysis.
- Basics of Security Information and Event Management (SIEM).
- Practical skills in monitoring and identifying potential security threats.
